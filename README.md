## üìã Description

**Delicious** is a modern food delivery platform, streamlining the ordering process for users and providing powerful product and account management tools for administrators.

---

## üõ†Ô∏è Technologies Used

- **Flutter:** Main UI framework
- **Firebase:** Authentication, Database, and Storage
- **GetX:** State management and navigation
- **Google Maps Plugin:** Delivery location and tracking
- **SSLCommerz:** Integrated payment solution

---

## üçî Features

- **User Operations:**
  - Multiple account creation methods via Firebase
  - Fetch and display dynamic food data from Firebase
  - Add to cart, manage favorites, filter products
  - Place orders seamlessly
  - Location tracking with Google Maps

- **Admin Panel:**
  - Control and manage user accounts
  - Add new products and edit existing product details

- **Payments:**
  - Secure and easy payment integration with SSLCommerz

## üöÄ Getting Started

### 1. **Clone the repository**
```bash
git clone https://github.com/md-ruhulamin/delicious.git
cd delicious
```

### 2. **Install Flutter & set up device**
- Follow [Flutter installation instructions](https://flutter.dev/docs/get-started/install).
- Use `flutter doctor` to verify installation.

### 3. **Install dependencies**
```bash
flutter pub get
```

### 4. **Configure Firebase**
- Follow instructions in the repo to add your `google-services.json` and configure your Firebase project.

### 5. **Run the app**
- **Android/iOS:** Launch emulator or connect device & run:
  ```bash
  flutter run
  ```
- **Web:** For web, run:
  ```bash
  flutter run -d chrome
  ```
